# 我的AC CODE
``` cpp
#include<bits/stdc++.h>
using namespace std;
int main(){
int dp[51], in;
dp[0]=1;
cin >> in;
for(int i=1;i<=in;i++){
dp[i]=i+dp[i-1];
}
cout << dp[in] <<'\n';
return 0;
}
```
